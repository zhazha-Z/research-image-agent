# Dice、IoU、Precision 和 Recall 指标解释

## 适用场景

这些指标常用于细胞显微图像、组织图像和实验图像中的目标分割质量评估。它们适合在已经有 ground truth mask 和 prediction mask 的情况下比较分割结果与人工标注之间的一致性。

## 核心概念解释

Dice 衡量预测区域与真实标注区域的重叠程度，数值越接近 1 表示两者越一致。IoU 使用交集除以并集，通常比 Dice 更严格。Precision 关注预测为目标的像素或区域中有多少是真正目标，Recall 关注真实目标中有多少被预测出来。

这些指标都需要 ground truth mask 和 prediction mask。没有真实标注时，不能计算真实 Dice、IoU、Precision 或 Recall，只能做定性描述或报告为待标注评估。

## 常见问题

- Dice 和 IoU 有什么区别：Dice 更强调重叠比例，IoU 对多分和漏分更敏感。
- 为什么 Dice 高但边界仍然不好：整体面积重叠高不代表边界每个位置都准确。
- 没有人工标注能不能计算指标：不能计算真实指标，只能使用估计、质控或人工复核。

## 局限性

单个指标不能完整反映分割质量。小目标、边界模糊、类别不平衡和标注差异都会影响指标解释。科研报告中应说明标注来源、评价区域和计算方式。

- Dice 和 IoU 有什么区别？
- Precision 和 Recall 分别说明什么？
- 没有 ground truth mask 能算真实 Dice 吗？
